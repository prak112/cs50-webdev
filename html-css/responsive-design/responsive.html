Welcome to Responsive Design

This page consists of content oriented for different wide and narrow window width sizes.

Responsive Design is controlled by CSS Query- @media. The @media queries could be used for multiple purposes.

For example : In this webpage, we used it to chnage the background color when the width of the window shrinks. Check it out!

Any CSS property can be controlled using @media Query. Such as some of the following :_

  1. Spacing or Padding based on screen sizes of devices
  2. Display or Hide certain content based on screen size
