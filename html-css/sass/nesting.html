Styling of Nested div elements

Task:

Create text styling for elements INSIDE div tag using SCSS

Order of events :

  1. Create HTML with nested elements Inside and Outside div tag
  2. Create SCSS with nested styling, check nesting.scss file
  3. Compile SCSS to CSS using, 
    sass filename.scss:filename.css
  4. Activate watch function using, 
    sass --watch filename.scss:filename.css
